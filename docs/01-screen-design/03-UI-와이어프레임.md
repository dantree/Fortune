| 항목 | 내용 |
|------|------|
| 버전 | v0.1 |
| 작성일 | 2026-07-05 |
| 기준 뷰포트 | 모바일 390×844 (iPhone 14 기준) |
| 그리드 | 16px 좌우 패딩, 8px spacing unit |

---

## 1. 디자인 원칙

1. **모바일 퍼스트** — SNS 유입 80% 이상 가정
2. **한 화면 한 목적** — 입력 / 기대 / 결과 분리
3. **광고 = 레이아웃 일부** — 슬롯 min-height 사전 확보로 CLS 방지
4. **터치 타겟** — 최소 44×44px
5. **가독성** — 본문 16px, 점수 64px Bold

## 7. 반응형 breakpoint

| breakpoint | width | 변경 사항 |
|------------|-------|----------|
| mobile | < 768px | 단일 컬럼, full-width CTA |
| tablet | 768~1024px | max-width 480px center |

## 8. iframe (Blogspot) 임베드 모드

```
┌ Blogspot 글 본문 ─────────────────┐
│  (블로그 텍스트...)                │
│  ┌─────────────────────────────┐  │
│  │  iframe: saju.kaltaelee.com │  │  height: 720px → postMessage auto
│  │  (SCR-00~02 동일 UI)         │  │
│  └─────────────────────────────┘  │
│  (블로그 텍스트...)                │
└───────────────────────────────────┘
```

| iframe 모드 | 변경 |
|-------------|------|
| `?embed=1` | ads-top 숨김, FTR 간소화 |
| postMessage | `resize` → 부모 iframe height 조정 |
