# 사주칼퇴리 계산 기준

> 엔진 버전: `SajuEngine` v1.2.0 (+ `SolarTerms`)  
> 원칙: **결정론(Deterministic)** — 같은 입력은 항상 같은 결과.

## 지원 범위

| 항목 | 지원 | 비고 |
|------|------|------|
| 양력 | ✅ | `YYYY-MM-DD` |
| 음력 | ✅ | KARI(한국천문연구원) 기준, 윤달 지원 |
| 일주 | ✅ | 환산된 양력으로 JDN 만세력 |
| 년주·띠 | ✅ | 태양 황경 **입춘 절입(KST)**. 미로드 시 2/4 간이 |
| 월주 | ✅ | 12절 절입 + 연간 오호둔(年上起月) |
| 대운 | ✅ | 성별·연간 음양 → 순/역행, 절입 일수÷3 기산 |
| 시주 | ✅ | 2시간 시진 + 오서둔 |
| 오늘의 운세 | ✅ | 내 일간 × 오늘 일주 |
| 추천 로또 | ✅ | 날짜(+생일) 시드, 오락용 |
| 별자리 | ✅ | 양력 생일 황도 12궁 |
| 오늘의 총운 | ✅ | 사주+띠+별자리 종합 |
| 풀사주 | ✅ | 년·월·일·시 + 대운 + 나이별 생활 가이드 |
| 생활 가이드 | ✅ | 현재/다음 대운, 5년 세운, 방위·색·보완 |

## 절기 (SolarTerms)

- 태양 시황경: Jean Meeus *Astronomical Algorithms* 근사
- 절입: 황경이 목표값(입춘 315° 등)에 도달하는 시각을 이진 탐색
- 표시·비교 기준: **한국 표준시(KST, UTC+9)**
- 출생 시각이 없으면 절기 비교에 **12:00 KST** 사용
- 입춘 당일이라도 절입 **이전**이면 이전 사주년 (예: 2000-02-04 정오 → 아직 토끼해)

## 월주

- 월을 가르는 **12절**(중기 제외): 입춘→寅 … 소한→丑
- 월간: 오호둔 — 甲己년 丙寅월 시작, 乙庚→戊寅, 丙辛→庚寅, 丁壬→壬寅, 戊癸→甲寅

## 대운

- 양남·음녀 → **순행**(다음 절입까지 일수÷3)
- 음남·양녀 → **역행**(직전 절입부터 일수÷3)
- 월주에서 ±1씩 10개 대운(각 10년)
- 세운·신살·정밀 명리 해석은 미포함

## 음력 → 양력

- 라이브러리: `korean-lunar-calendar` (KARI)
- 범위: 음력 1000~2050 / 양력 1000~2050
- 사주·오행 계산은 **항상 양력**으로 수행
- 음력 입력 시 화면에 양력 환산일을 함께 표시
- 중국력과 일부 날짜가 다를 수 있음 (한국 기준 사용)

## 시주

- 시진: 자(23~01) … 해(21~23), 2시간 단위
- **23:00 이후 자시** → 일주를 **다음날** 기준으로 계산
- 시천간: 오서둔 `(일간%5)×2 + 시진지지`

## 오늘의 운세 / 로또

- 운세: 내 일간 오행 ↔ 오늘 일주 오행 상생·상극 + 고정 문구
- 로또: 1~45 중 6개 + 보너스, 같은 날·같은 생일이면 동일 번호
- **당첨 보장 없음** (오락·참고용)
