# 사주 도구 계산 기준

> 엔진 버전: `SajuEngine` v1.0.0  
> 원칙: **결정론(Deterministic)** — 같은 입력은 항상 같은 결과. 랜덤·외부 API·`Date.now()`를 계산에 사용하지 않음.

## 지원 범위

| 항목 | 지원 | 비고 |
|------|------|------|
| 양력(그레고리력) | ✅ | `YYYY-MM-DD` |
| 일주(日柱) | ✅ | JDN 만세력 공식 |
| 년주(年柱) | ✅ | 입춘 간이 기준 |
| 띠(地支) | ✅ | 년주 지지 기준 |
| 시주(時柱) | ❌ | v2 예정 |
| 음력 | ❌ | 양력만 |
| 월주(月柱) | ❌ | v2 예정 |

## 일주 계산

1. 입력 날짜를 율리우스일(JDN)로 변환
2. `일주 인덱스 = (JDN + 49) % 60`
3. 인덱스를 천간(10)·지지(12)로 분해 → 60갑자

**검증 앵커:** `2000-01-01` → `戊午` (일주 54)

```text
JDN 공식: 표준 그레고리력 알고리즘
오프셋 49: 2000-01-01 = 戊午 기준 역산
```

## 년주·띠 계산 (입춘)

**간이 입춘 규칙:**

- **2월 4일 이전** → 전년도 사주년
- **2월 4일 이후(포함)** → 해당 연도 사주년

예시:

| 양력 날짜 | 사주년 | 띠 |
|-----------|--------|-----|
| 2000-01-01 | 1999 | 토끼 (己卯) |
| 2000-02-03 | 1999 | 토끼 |
| 2000-02-04 | 2000 | 용 (庚辰) |
| 2000-02-05 | 2000 | 용 |

년주 인덱스: `(사주년 - 4) % 60`

> ⚠️ 실제 입춘 시각은 매년 달라집니다. 본 도구는 **2월 4일 고정** 간이 규칙을 사용합니다. 전문 만세력과 ±1일 차이가 날 수 있습니다.

## 오행

일간(天干) 기준 오행:

| 천간 | 오행 |
|------|------|
| 甲·乙 | 목(木) |
| 丙·丁 | 화(火) |
| 戊·己 | 토(土) |
| 庚·辛 | 금(金) |
| 壬·癸 | 수(水) |

상생·상극 점수는 `OHENG_REL` 5×5 고정 행렬 사용.

## MBTI × 오행 궁합

- MBTI 그룹 → 오행: NT→금, NF→화, SJ→토, SP→수
- 종합 점수: 오행 60% + MBTI 40%
- MBTI 16×16 매트릭스: `js/mbti-compat.js` 고정값 (랜덤·charCodeAt 없음)

## 띠 궁합

- `js/zodiac-compat.js` 12×12 고정표
- 삼합·육합·충·동일·보통 관계별 고정 점수

## 검증

브라우저에서 `/tests/verify.html` 또는 Node:

```bash
node -e "var w={};eval(require('fs').readFileSync('js/saju-engine.js','utf8'));console.log(w.SajuEngine.runSelfTest())"
```

고정 검증 케이스는 `SajuEngine.VERIFY_CASES` 참조.

## 제한 사항

- 시주 미지원으로 **자시(23~01시) 일주 변경** 미반영
- 음력·윤달 미지원
- 입춘 시각 미반영 (2/4 고정)
- 전문 역술가 만세력과 완전 일치를 보장하지 않음
